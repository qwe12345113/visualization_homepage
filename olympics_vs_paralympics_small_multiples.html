Olympics vs Paralympics: Small-Multiples Comparison

This visualisation is intended to compare how long-term team performance trends differ between the Olympics and the Paralympics. Each small-multiple panel represents one country. Within each panel, two lines are shown: one for Olympic performance and one for Paralympic performance, both measured using the same weighted medal score. Consistent scales across panels support country-to-country comparison.
